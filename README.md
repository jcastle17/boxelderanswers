# Box Elder Deserves Answers

**Box Elder Deserves Answers** is a public information and community-support website connected to the petition:

**Pause Project Stratos Now — Box Elder Deserves Answers**  
Petition: https://www.change.org/p/pause-project-stratos-now-box-elder-deserves-answers

This repository contains the public website files for **boxelderanswers.com**.

---

## Purpose

This site was created to help organize public attention, share public-facing resources, and support responsible review before major decisions affecting Box Elder County move forward.

The main public position is simple:

> **Pause Project Stratos long enough to release records, allow independent review, and give residents a meaningful chance to be heard before irreversible action moves forward.**

---

## Core Concerns Raised

The petition and related public messaging focus on the need for more transparency and review regarding:

- Water use
- Aquifer impacts
- Great Salt Lake impacts
- Natural gas power generation
- Emissions and environmental review
- Public health concerns
- Tax incentives and public costs
- Local control and public process
- Public release of agreements, water-related documents, tax documents, and project records
- A meaningful public meeting and opportunity for residents to respond

The purpose of the site is not to make final findings for the public. The purpose is to push for answers, records, independent review, and public participation.

---

## What This Website Does

This website may be used to:

- Direct residents to the Change.org petition
- Explain the public request to pause Project Stratos
- Organize community awareness
- Share official public resources
- Help residents understand where to report serious safety concerns
- Keep the public message consistent and accessible

## Community Safety Resources

The site includes a **Community Safety Resources** section.

That section is designed to send people to official agencies, not to collect private allegations through this website.

If someone is in immediate danger, call **911**.

For suspected child abuse or neglect in Utah, use official reporting channels such as:

- Utah DCFS Child Protective Services
- Utah Child Abuse/Neglect Hotline: **1-855-323-DCFS (3237)**
- Utah DCFS official reporting form

For public registry information, the site directs users to the official Utah offender registry resources. Registry information should be used for safety awareness only and should not be used to harass, threaten, stalk, intimidate, or encourage vigilante action.
